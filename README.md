Punto de Equilibrio: Definición de las Mecánicas y Estructuras de Datos

- Punto de Equilibrio es un juego de cartas single player que se centra en mover cartas y tomar decisiones estratégicas para mantener el equilibrio en la mesa.
- El objetivo es mover cartas de una columna a otra, creando secuencias y combinaciones para avanzar en el juego.
- El jugador debe gestionar un límite de movimientos y tomar decisiones estratégicas para evitar bloquear cartas y mantener el equilibrio.
- El juego cuenta con un sistema de puntos que se otorgan al mover cartas a la fundación, y el objetivo es maximizar la puntuación.

Estructuras de Datos:

- Lista Enlazada Simple: Se utiliza para representar el mazo de cartas, permitiendo agregar y eliminar cartas de manera eficiente.
- Lista Enlazada Doble: Se utiliza para representar las columnas de cartas, permitiendo mover cartas hacia adelante y hacia atrás de manera eficiente.
- Lista Circular Simple: Se utiliza para implementar el menú del juego, permitiendo al jugador navegar entre las diferentes opciones de manera circular.
- Lista Circular Doble: Se utiliza para implementar la función de "deshacer" y "rehacer", permitiendo al jugador revertir y reintentar movimientos de manera eficiente.
- Pila: Se utiliza para representar la fundación, permitiendo agregar y eliminar cartas de manera eficiente. También se utiliza para implementar la función de "deshacer" y "rehacer".
- Pila de Historial: Se utiliza para almacenar los movimientos del jugador, permitiendo al jugador revertir y reintentar movimientos de manera eficiente.

Mecánicas Principales:

- Mover cartas entre columnas para crear secuencias y combinaciones
- Gestionar un límite de movimientos para evitar bloquear cartas
- Tomar decisiones estratégicas para mantener el equilibrio en la mesa
- Maximizar la puntuación moviendo cartas a la fundación
- Utilizar la función de "deshacer" y "rehacer" para revertir y reintentar movimientos
